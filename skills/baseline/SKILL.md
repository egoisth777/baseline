# baseline

Use the baseline manager from this plugin root instead of delegating to a root
`SKILL.md`. In a source checkout, the plugin root is two directories above this
file and contains `scripts/manage.js`, `.codex-plugin/plugin.json`, and
`.claude-plugin/plugin.json`.

Common commands from the plugin root:

    node scripts/manage.js status
    node scripts/manage.js install
    node scripts/manage.js update
    node scripts/manage.js verify
    node scripts/manage.js doctor
    node scripts/manage.js doctor --fix
    node scripts/manage.js uninstall

Installed control surfaces generated by `install` point at the self-contained
manager artifacts under `BASELINE_HOME` (else `~/.baseline`) and Codex activation
uses `$CODEX_HOME/plugins/cache/baseline/baseline/local` plus the empty
`[plugins."baseline@baseline"]` table in `$CODEX_HOME/config.toml`.
